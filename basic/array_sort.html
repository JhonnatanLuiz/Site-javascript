JavaScript Array Sort

Ordenação Alfabética

  • sort() - Ordena o array
  • reverse() - Inverte o array
  • toSorted() - Ordena sem modificar
  • toReversed() - Inverte sem modificar

Ordenação Numérica

  • Função de comparação
  • Ordenação aleatória
  • Math.min() / Math.max()
  • Ordenando objetos

Ordenando um Array

O método sort() ordena um array alfabeticamente:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
frutas.sort();
// ["Banana", "Laranja", "Manga", "Maçã"]

Invertendo um Array

O método reverse() inverte os elementos de um array:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
frutas.reverse();
// ["Manga", "Maçã", "Laranja", "Banana"]

Combinando sort() e reverse(), você pode ordenar um array em ordem decrescente:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
frutas.sort();
frutas.reverse();
// ["Maçã", "Manga", "Laranja", "Banana"]

toSorted() e toReversed() (ES2023)

O ES2023 adicionou o método toSorted() como uma forma segura de ordenar um array sem alterar o original:

const meses = ["Jan", "Fev", "Mar", "Abr"];
const ordenado = meses.toSorted();
// meses: ["Jan", "Fev", "Mar", "Abr"] (inalterado)
// ordenado: ["Abr", "Fev", "Jan", "Mar"]

Da mesma forma, toReversed() inverte sem alterar o original:

const meses = ["Jan", "Fev", "Mar", "Abr"];
const invertido = meses.toReversed();
// meses: ["Jan", "Fev", "Mar", "Abr"] (inalterado)
// invertido: ["Abr", "Mar", "Fev", "Jan"]

Diferença: toSorted() e toReversed() criam um novo array mantendo o original inalterado, enquanto sort() e reverse() modificam o array original.

Ordenação Numérica

Por padrão, a função sort() ordena valores como strings. Isso funciona bem para strings ("Apple" vem antes de "Banana").

Problema: Se números são ordenados como strings, "25" é maior que "100", porque "2" é maior que "1". Por isso, o método sort() produz resultado incorreto ao ordenar números.

Você pode corrigir isso fornecendo uma função de comparação:

const pontos = [40, 100, 1, 5, 25, 10];

// Ordenação CRESCENTE
pontos.sort(function(a, b) { return a - b });
// [1, 5, 10, 25, 40, 100]

// Ordenação DECRESCENTE
pontos.sort(function(a, b) { return b - a });
// [100, 40, 25, 10, 5, 1]

A Função de Comparação

O objetivo da função de comparação é definir uma ordem de classificação alternativa. A função deve retornar um valor negativo, zero ou positivo:

function(a, b) { return a - b }
  • • Se o resultado é negativo, a é ordenado antes de b
  • • Se o resultado é positivo, b é ordenado antes de a
  • • Se o resultado é 0, não há mudança na ordem

Exemplo: Ao comparar 40 e 100, a função calcula 40 - 100 = -60. Como é negativo, 40 será ordenado antes de 100.

Ordenação Aleatória

Você pode ordenar um array em ordem aleatória usando a função de ordenação:

const pontos = [40, 100, 1, 5, 25, 10];
pontos.sort(function() { return 0.5 - Math.random() });

Nota: O método acima não é muito preciso e pode favorecer alguns números. O método mais correto é o Fisher Yates shuffle.

O Método Fisher Yates

O método mais popular e correto, introduzido na ciência de dados em 1938:

const pontos = [40, 100, 1, 5, 25, 10];

for (let i = pontos.length - 1; i > 0; i--) {
    let j = Math.floor(Math.random() * (i + 1));
    let k = pontos[i];
    pontos[i] = pontos[j];
    pontos[j] = k;
}

Encontrar Menor e Maior Valor

Não existem funções nativas para encontrar o menor ou maior valor em um array. Você tem 3 opções:

1. Usando sort()

const pontos = [40, 100, 1, 5, 25, 10];

// Ordenar e pegar o primeiro/último
pontos.sort(function(a, b) { return a - b });
let menor = pontos[0];              // 1
let maior = pontos[pontos.length - 1];  // 100

2. Usando Math.min() e Math.max()

const pontos = [40, 100, 1, 5, 25, 10];

// Usando spread operator (ES6)
let menor = Math.min(...pontos);  // 1
let maior = Math.max(...pontos);  // 100

// Usando apply
let menor2 = Math.min.apply(null, pontos);  // 1
let maior2 = Math.max.apply(null, pontos);  // 100

3. Função personalizada

function encontrarMin(arr) {
    let len = arr.length;
    let min = Infinity;
    while (len--) {
        if (arr[len] < min) {
            min = arr[len];
        }
    }
    return min;
}

function encontrarMax(arr) {
    let len = arr.length;
    let max = -Infinity;
    while (len--) {
        if (arr[len] > max) {
            max = arr[len];
        }
    }
    return max;
}

Ordenando Arrays de Objetos

Arrays JavaScript frequentemente contêm objetos. Mesmo se os objetos tiverem propriedades de diferentes tipos, o método sort() pode ser usado:

const carros = [
    { tipo: "Volvo", ano: 2016 },
    { tipo: "Saab", ano: 2001 },
    { tipo: "BMW", ano: 2010 }
];

// Ordenar por ano (numérico)
carros.sort(function(a, b) { return a.ano - b.ano });

// Ordenar por tipo (string)
carros.sort(function(a, b) {
    let x = a.tipo.toLowerCase();
    let y = b.tipo.toLowerCase();
    if (x < y) { return -1; }
    if (x > y) { return 1; }
    return 0;
});

Ordenação Estável (ES2019)

O ES2019 revisou o método sort(). Antes de 2019, a especificação permitia algoritmos de ordenação instáveis como QuickSort.

Após ES2019, os navegadores devem usar um algoritmo de ordenação estável: ao ordenar elementos por um valor, os elementos devem manter sua posição relativa a outros elementos com o mesmo valor.

const meuArr = [
    { nome: "X00", preco: 100 },
    { nome: "X01", preco: 100 },
    { nome: "X02", preco: 100 },
    { nome: "X03", preco: 100 },
    { nome: "X04", preco: 110 },
    { nome: "X05", preco: 110 }
];

// Ao ordenar por preço, X00-X03 mantêm sua ordem relativa
// e X04-X05 também mantêm sua ordem relativa

Ordenação estável: Elementos com valores iguais mantêm sua posição original relativa após a ordenação.
