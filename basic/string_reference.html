JavaScript String Reference

Referência completa de métodos e propriedades de String.

Método Descrição
charAt() Retorna o caractere em um índice especificado (posição)
charCodeAt() Retorna o Unicode do caractere em um índice especificado
concat() Junta duas ou mais strings e retorna uma nova string
endsWith() Verifica se uma string termina com uma string especificada
includes() Verifica se uma string contém uma string especificada
indexOf() Retorna o índice da primeira ocorrência de um valor especificado em uma string
lastIndexOf() Retorna o índice da última ocorrência de um valor especificado em uma string
length Retorna o comprimento de uma string
match() Pesquisa uma string por uma correspondência contra uma expressão regular
repeat() Retorna uma nova string com um número especificado de cópias de uma string existente
replace() Pesquisa uma string por um valor especificado, ou uma expressão regular, e retorna uma nova string onde os valores especificados são substituídos
search() Pesquisa uma string por um valor especificado, ou expressão regular, e retorna a posição da correspondência
slice() Extrai uma parte de uma string e retorna uma nova string
split() Divide uma string em uma matriz de substrings
startsWith() Verifica se uma string começa com caracteres especificados
substr() Extrai os caracteres de uma string, começando em uma posição inicial especificada, e através do número especificado de caracteres
substring() Extrai os caracteres de uma string, entre dois índices especificados
toLowerCase() Converte uma string para letras minúsculas
toUpperCase() Converte uma string para letras maiúsculas
trim() Remove espaços em branco de ambas as extremidades de uma string
