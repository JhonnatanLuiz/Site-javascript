JavaScript Arrays

Arrays JavaScript são usados para armazenar múltiplos valores em uma única variável.

O que é um Array?

Um array é uma variável especial, que pode conter mais de um valor:

JavaScript 
const cars = ["Saab", "Volvo", "BMW"];
const cars = ["Saab", "Volvo", "BMW"];

Criando um Array

Usar um literal de array é a maneira mais fácil de criar um array JavaScript.

Sintaxe:

const array_name = [item1, item2, ...];

É uma prática comum declarar arrays com a palavra-chave const.

Acessando Elementos

Você acessa um elemento de array referindo-se ao número do índice.

Nota: Os índices de array começam com 0. [0] é o primeiro elemento. [1] é o segundo elemento.

JavaScript 
const cars = ["Saab", "Volvo", "BMW"];
let car = cars[0];
const cars = ["Saab", "Volvo", "BMW"];
let car = cars[0];

Alterando Elementos

Esta instrução altera o valor do primeiro elemento em cars:

JavaScript 
cars[0] = "Opel";
cars[0] = "Opel";
