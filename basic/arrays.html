JavaScript Arrays

Um array é uma variável especial que pode conter mais de um valor.

Exemplo

const cars = ["Saab", "Volvo", "BMW"];

Por que Usar Arrays?

Se você tem uma lista de itens (por exemplo, uma lista de nomes de carros), armazenar os nomes em variáveis separadas ficaria assim:

let car1 = "Saab";
let car2 = "Volvo";
let car3 = "BMW";

Mas e se você quisesse percorrer os carros e encontrar um específico? E se você tivesse não 3 carros, mas 300?

A solução é um array!

Um array pode conter muitos valores sob um único nome, e você pode acessar os valores referenciando um número de índice.

Criando um Array

A maneira mais fácil de criar um array JavaScript é usando uma literal de array:

const array_name = [item1, item2, ...];

💡 Nota: É uma prática comum declarar arrays com a palavra-chave const.

Exemplo - Criando Array com Valores

const cars = ["Saab", "Volvo", "BMW"];

Você também pode criar um array vazio e adicionar elementos depois:

Exemplo - Array Vazio

const cars = [];
cars[0] = "Saab";
cars[1] = "Volvo";
cars[2] = "BMW";

Acessando Elementos do Array

Você acessa um elemento do array referenciando o número do índice:

Exemplo

const cars = ["Saab", "Volvo", "BMW"];
let car = cars[0]; // Acessa o primeiro elemento

⚠️ Índices de array começam com 0.

[0] é o primeiro elemento. [1] é o segundo elemento.

Alterando um Elemento do Array

Esta instrução altera o valor do primeiro elemento em cars:

Exemplo

const cars = ["Saab", "Volvo", "BMW"];
cars[0] = "Opel"; // Altera "Saab" para "Opel"

Convertendo um Array para String

O método toString() converte um array em uma string de valores separados por vírgulas:

Exemplo

const fruits = ["Banana", "Laranja", "Maçã", "Manga"];
document.getElementById("demo").innerHTML = fruits.toString();
// Resultado: Banana,Laranja,Maçã,Manga

Arrays são Objetos

Arrays são um tipo especial de objetos. O operador typeof em JavaScript retorna "object" para arrays.

Mas, arrays JavaScript são melhor descritos como arrays.

Exemplo - typeof

const cars = ["Saab", "Volvo", "BMW"];
typeof cars; // Retorna "object"

💡 Arrays usam números para acessar seus elementos. Objetos usam nomes para acessar seus "membros".

A Propriedade length

A propriedade length retorna o comprimento de um array (número de elementos):

Exemplo

const fruits = ["Banana", "Laranja", "Maçã", "Manga"];
let length = fruits.length; // Retorna 4

💡 A propriedade length é sempre um a mais que o maior índice do array.

Acessando Primeiro e Último Elemento

Primeiro Elemento

let first = fruits[0];

Último Elemento

let last = fruits[fruits.length - 1];

Percorrendo Elementos do Array

Uma maneira de percorrer um array é usando um loop for:

Exemplo - Loop for

const fruits = ["Banana", "Laranja", "Maçã", "Manga"];
let fLen = fruits.length;
let text = "<ul>";

for (let i = 0; i < fLen; i++) {
    text += "<li>" + fruits[i] + "</li>";
}
text += "</ul>";

Você também pode usar a função Array.forEach():

Exemplo - forEach()

const fruits = ["Banana", "Laranja", "Maçã", "Manga"];
let text = "<ul>";

fruits.forEach(myFunction);
text += "</ul>";

function myFunction(value) {
    text += "<li>" + value + "</li>";
}

Adicionando Elementos ao Array

A maneira mais fácil de adicionar um novo elemento a um array é usando o método push():

Exemplo - push()

const fruits = ["Banana", "Laranja", "Maçã"];
fruits.push("Limão"); // Adiciona "Limão" ao final

Novo elemento também pode ser adicionado usando a propriedade length:

Exemplo - Usando length

const fruits = ["Banana", "Laranja", "Maçã"];
fruits[fruits.length] = "Limão"; // Adiciona "Limão" ao final

⚠️ AVISO!

Adicionar elementos com índices altos pode criar "buracos" indefinidos em um array:

fruits[6] = "Limão"; // Cria "buracos" undefined

Como Reconhecer um Array

Uma pergunta comum é: Como eu sei se uma variável é um array?

O problema é que o operador typeof retorna "object":

Solução 1: Array.isArray()

Array.isArray(fruits); // true

Solução 2: instanceof

fruits instanceof Array; // true

Arrays Aninhados

Valores em objetos podem ser arrays, e valores em arrays podem ser objetos:

Exemplo

const myObj = {
    name: "John",
    age: 30,
    cars: [
        {name: "Ford", models: ["Fiesta", "Focus", "Mustang"]},
        {name: "BMW", models: ["320", "X3", "X5"]},
        {name: "Fiat", models: ["500", "Panda"]}
    ]
};
