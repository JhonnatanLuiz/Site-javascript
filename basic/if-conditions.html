JavaScript Conditionals

Declarações condicionais são usadas para executar ações diferentes com base em diferentes condições.

Declarações Condicionais

Frequentemente ao escrever código, você quer executar ações diferentes para decisões diferentes.

Você pode usar declarações condicionais em seu código para fazer isso.

Em JavaScript temos as seguintes declarações condicionais:

  • Use if para especificar um bloco de código a ser executado, se uma condição especificada for verdadeira
  • Use else para especificar um bloco de código a ser executado, se a mesma condição for falsa
  • Use else if para especificar uma nova condição a testar, se a primeira condição for falsa
  • Use switch para especificar muitos blocos alternativos de código a serem executados

A Declaração if

Use a declaração if para especificar um bloco de código JavaScript a ser executado se uma condição for verdadeira.

Sintaxe

if (condição) {
    // bloco de código a ser executado se a condição for verdadeira
}

💡 Note que if está em letras minúsculas. Letras maiúsculas (If ou IF) gerarão um erro JavaScript.

Exemplo

if (hora < 18) {
    saudacao = "Bom dia";
}

A Declaração else

Use a declaração else para especificar um bloco de código a ser executado se a condição for falsa.

Sintaxe

if (condição) {
    // bloco executado se a condição for verdadeira
} else {
    // bloco executado se a condição for falsa
}

A Declaração else if

Use a declaração else if para especificar uma nova condição se a primeira condição for falsa.

Sintaxe

if (condição1) {
    // bloco executado se condição1 for verdadeira
} else if (condição2) {
    // bloco executado se condição2 for verdadeira
} else {
    // bloco executado se ambas forem falsas
}
