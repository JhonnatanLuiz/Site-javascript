JavaScript Number Reference

Referência Completa de Number

Abaixo está uma lista completa de propriedades e métodos do objeto Number.

Nome Descrição
constructor Retorna a função que criou o protótipo do objeto Number
EPSILON A diferença entre 1 e o menor número > 1
isFinite() Verifica se um valor é um número finito
isInteger() Verifica se um valor é um inteiro
isNaN() Verifica se um valor é Number.NaN
isSafeInteger() Verifica se um valor é um inteiro seguro
MAX_SAFE_INTEGER O inteiro seguro máximo em JavaScript (2^53 - 1)
MIN_SAFE_INTEGER O inteiro seguro mínimo em JavaScript -(2^53 - 1)
MAX_VALUE O maior número possível em JavaScript
MIN_VALUE O menor número possível em JavaScript
NaN "Not-a-Number" (Não é um número)
NEGATIVE_INFINITY Infinito negativo
POSITIVE_INFINITY Infinito positivo
parseFloat() Analisa uma string e retorna um número
parseInt() Analisa uma string e retorna um inteiro
prototype Permite adicionar propriedades e métodos a um objeto
toExponential() Retorna uma string, com um número arredondado e escrito usando notação exponencial
toFixed() Retorna uma string, com o número escrito com um número especificado de decimais
toLocaleString() Retorna um número como uma string, usando configurações locais
toPrecision() Retorna uma string, com um número escrito com um comprimento especificado
toString() Retorna um número como uma string
valueOf() Retorna um número como um número
