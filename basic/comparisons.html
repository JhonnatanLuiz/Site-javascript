JavaScript Comparison Operators

Operadores de comparação são usados em declarações lógicas para determinar igualdade ou diferença entre variáveis ou valores.

Dado que x = 5, a tabela abaixo explica os operadores de comparação:

x == 5		 false

Como Usar Operadores de Comparação?

Operadores de comparação podem ser usados em declarações condicionais para comparar valores e tomar ações dependendo do resultado:

Exemplo

if (idade < 18) {
    texto = "Muito jovem";
}

Você aprenderá mais sobre o uso de declarações condicionais no próximo capítulo deste tutorial.

Igual a (==)

O operador de igualdade (==) compara se dois valores são iguais:

Exemplo

let x = 5;
x == 8   // retorna false
x == 5   // retorna true
x == "5" // retorna true

Resultado:

Igual em Valor e Tipo (===)

O operador de igualdade estrita (===) compara se dois valores são iguais E do mesmo tipo:

Exemplo

let x = 5;
x === 5   // retorna true
x === "5" // retorna false

Resultado:

⚠️ Atenção: É comum ver o operador de igualdade dupla (==) em vez do triplo (===). Eles funcionam de forma diferente! O === compara tipo E valor, enquanto == compara apenas o valor (fazendo conversão de tipo se necessário).

Diferente (!=)

O operador de desigualdade (!=) compara se dois valores são diferentes:

Exemplo

let x = 5;
x != 8  // retorna true

Resultado:

Diferente em Valor ou Tipo (!==)

O operador de desigualdade estrita (!==) compara se dois valores são diferentes OU de tipos diferentes:

Exemplo

let x = 5;
x !== 5   // retorna false
x !== "5" // retorna true
x !== 8   // retorna true

Resultado:

Maior Que (>)

O operador maior que (>) retorna true se o valor da esquerda for maior que o da direita:

Exemplo

let x = 5;
x > 8  // retorna false
x > 3  // retorna true

Resultado:

Menor Que (<)

O operador menor que (<) retorna true se o valor da esquerda for menor que o da direita:

Exemplo

let x = 5;
x < 8  // retorna true
x < 3  // retorna false

Resultado:

Maior ou Igual (>=)

O operador maior ou igual (>=) retorna true se o valor da esquerda for maior ou igual ao da direita:

Exemplo

let x = 5;
x >= 8  // retorna false
x >= 5  // retorna true

Resultado:

Menor ou Igual (<=)

O operador menor ou igual (<=) retorna true se o valor da esquerda for menor ou igual ao da direita:

Exemplo

let x = 5;
x <= 8  // retorna true
x <= 3  // retorna false

Resultado:

Operador Ternário (?)

JavaScript também contém um operador condicional que atribui um valor a uma variável com base em alguma condição.

Sintaxe

let nomeVariavel = (condição) ? valor1 : valor2;

Exemplo

let idade = 18;
let podeVotar = (idade < 18) ? "Muito jovem" : "Velho o suficiente";

Comparando Tipos Diferentes

Comparar dados de tipos diferentes pode dar resultados inesperados.

Ao comparar uma string com um número, JavaScript converterá a string em um número ao fazer a comparação. Uma string vazia é convertida para 0. Uma string não numérica é convertida para NaN, que sempre retorna false.

Caso Valor
2 < 12 true
2 < "12" true
2 < "João" false
2 > "João" false
2 == "João" false
"2" < "12" false
"2" > "12" true
"2" == "12" false

⚠️ Importante: Ao comparar duas strings, "2" será maior que "12", porque (alfabeticamente) 1 é menor que 2.

Para garantir um resultado adequado, as variáveis devem ser convertidas para o tipo apropriado antes da comparação.
