JavaScript Operators

Os operadores JavaScript são usados para realizar operações em variáveis e valores.

Exemplo

let x = 5;
let y = 2;
let z = x + y;  // z = 7

Resultado:

Tipos de Operadores JavaScript

Existem diferentes tipos de operadores JavaScript:

  • Operadores Aritméticos (+, -, *, /, %, ++, --)
  • Operadores de Atribuição (=, +=, -=, *=, /=, %=)
  • Operadores de Comparação (==, ===, !=, !==, >, <, >=, <=)
  • Operadores Lógicos (&&, ||, !)
  • Operadores de String (+)
  • Operadores Ternários (? :)
  • Operadores de Tipo (typeof, instanceof)

Operadores Aritméticos

Operadores aritméticos realizam operações matemáticas em números:

Operador Descrição Exemplo
+ Adição x + y
- Subtração x - y
* Multiplicação x * y
/ Divisão x / y
% Módulo (resto) x % y
++ Incremento x++
-- Decremento x--
** Exponenciação x ** y

Operadores de Atribuição

Operadores de atribuição atribuem valores a variáveis JavaScript:

Operador Exemplo Equivalente a
= x = y x = y
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y

💡 Dica: Para saber mais sobre operadores de atribuição, visite a página JS Assignment.

Operadores de Comparação

Operadores de comparação são usados em instruções lógicas para determinar igualdade ou diferença entre variáveis ou valores:

Operador Descrição Exemplo
== Igual a x == y
=== Valor igual e tipo igual x === y
!= Diferente x != y
!== Valor diferente ou tipo diferente x !== y
> Maior que x > y
< Menor que x < y
>= Maior ou igual a x >= y
<= Menor ou igual a x <= y

💡 Dica: Para saber mais sobre operadores de comparação, visite a página JS Comparisons.

Operadores Lógicos

Operadores lógicos são usados para determinar a lógica entre variáveis ou valores:

Operadores Lógicos

  • &&
    AND lógico: Retorna true se ambas as expressões forem verdadeiras 
    let resultado = (x > 5 && y < 10);
  • ||
    OR lógico: Retorna true se pelo menos uma das expressões for verdadeira 
    let resultado = (x > 5 || y < 10);
  • !
    NOT lógico: Inverte o valor booleano 
    let resultado = !(x > 5);

Operador de String

O operador + também pode ser usado para concatenar (juntar) strings:

Exemplo

let texto1 = "João";
let texto2 = "Silva";
let nomeCompleto = texto1 + " " + texto2;  // "João Silva"

Operadores de Tipo

  • typeof
    Retorna o tipo de uma variável 
    typeof "João"  // Retorna "string"
  • instanceof
    Retorna true se um objeto é uma instância de um tipo de objeto 
    let data = new Date();
data instanceof Date  // Retorna true
