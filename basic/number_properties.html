JavaScript Number Properties

Propriedades de número pertencem ao objeto Number do JavaScript.

JavaScript EPSILON

Number.EPSILON é a diferença entre o menor número de ponto flutuante maior que 1 e 1.

Exemplo

let x = Number.EPSILON;

Resultado:

JavaScript MAX_VALUE e MIN_VALUE

Number.MAX_VALUE é uma constante que representa o maior número possível em JavaScript.

Number.MIN_VALUE é uma constante que representa o menor número possível em JavaScript.

Exemplo

let x = Number.MAX_VALUE;
let y = Number.MIN_VALUE;

Resultado:

Inteiros Seguros

Number.MAX_SAFE_INTEGER representa o inteiro seguro máximo em JavaScript (2^53 - 1).

Number.MIN_SAFE_INTEGER representa o inteiro seguro mínimo em JavaScript -(2^53 - 1).

Exemplo

let x = Number.MAX_SAFE_INTEGER;
let y = Number.MIN_SAFE_INTEGER;
