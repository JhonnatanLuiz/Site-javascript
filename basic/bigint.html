JavaScript BigInt

Variáveis JavaScript BigInt são usadas para armazenar números inteiros grandes demais para serem representados por um Number normal do JavaScript.

Criando BigInt

Para criar um BigInt, adicione n ao final de um inteiro ou chame BigInt():

Exemplo

let x = 1234567890123456789012345n;
let y = BigInt("1234567890123456789012345");

Resultado:

Aritmética BigInt

Operadores como +, -, *, /, ** e % funcionam com BigInt.

Exemplo

let x = 9007199254740995n;
let y = 9007199254740995n;
let z = x * y;

Resultado:

Misturando Tipos

Você não pode misturar BigInt e Number em operações aritméticas. Você deve converter explicitamente.

Exemplo

let x = 5n;
let y = Number(x) / 2;

Resultado:

Typeof BigInt

O operador typeof retorna "bigint" para variáveis BigInt.

Exemplo
