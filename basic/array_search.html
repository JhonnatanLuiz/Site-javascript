JavaScript Array Search

Métodos de Busca em Arrays

  • indexOf() - Encontra a posição do primeiro elemento
  • lastIndexOf() - Encontra a posição do último elemento
  • includes() - Verifica se o elemento existe
  • find() - Retorna o valor do primeiro elemento encontrado
  • findIndex() - Retorna o índice do primeiro elemento encontrado
  • findLast() - Retorna o valor do último elemento encontrado
  • findLastIndex() - Retorna o índice do último elemento encontrado

JavaScript Array indexOf()

O método indexOf() busca um elemento no array e retorna sua posição:

const frutas = ["Maçã", "Laranja", "Maçã", "Manga"];
let posicao = frutas.indexOf("Maçã");  // Retorna 0

Nota: O primeiro item tem posição 0, o segundo tem posição 1, e assim por diante.

Sintaxe

array.indexOf(item, inicio)
item Obrigatório. O item a buscar.
inicio Opcional. Onde iniciar a busca. Valores negativos iniciam a partir do final.

indexOf() retorna -1 se o item não for encontrado. Se existir mais de uma ocorrência, retorna a posição da primeira.

JavaScript Array lastIndexOf()

O método lastIndexOf() é igual ao indexOf(), mas retorna a posição da última ocorrência do elemento:

const frutas = ["Maçã", "Laranja", "Maçã", "Manga"];
let posicao = frutas.lastIndexOf("Maçã");  // Retorna 2

Sintaxe

array.lastIndexOf(item, inicio)
item Obrigatório. O item a buscar.
inicio Opcional. Onde iniciar a busca. Valores negativos iniciam a partir do final e buscam em direção ao início.

JavaScript Array includes()

O ECMAScript 2016 introduziu o método includes(). Ele verifica se um elemento está presente no array (incluindo NaN, diferente de indexOf):

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
frutas.includes("Manga");  // true
frutas.includes("Uva");    // false

Sintaxe

array.includes(item-busca)

Nota: includes() permite verificar valores NaN, diferentemente de indexOf().

JavaScript Array find()

O método find() retorna o valor do primeiro elemento que passa em uma função de teste:

const numeros = [4, 9, 16, 25, 29];

// Encontra o primeiro número maior que 18
let primeiro = numeros.find(function(valor) {
    return valor > 18;
});
// Retorna 25

// Usando arrow function
let primeiro2 = numeros.find(valor => valor > 18);
// Retorna 25

A função de callback recebe 3 argumentos:

  • O valor do item
  • O índice do item
  • O próprio array
const numeros = [4, 9, 16, 25, 29];
let primeiro = numeros.find(function(valor, indice, array) {
    console.log(`Testando índice ${indice}: ${valor}`);
    return valor > 18;
});

JavaScript Array findIndex()

O método findIndex() retorna o índice do primeiro elemento que passa em uma função de teste:

const numeros = [4, 9, 16, 25, 29];

// Encontra o índice do primeiro número maior que 18
let indice = numeros.findIndex(function(valor) {
    return valor > 18;
});
// Retorna 3 (porque 25 está no índice 3)

// Usando arrow function
let indice2 = numeros.findIndex(valor => valor > 18);
// Retorna 3

A função de callback recebe os mesmos 3 argumentos: valor, índice e array.

JavaScript Array findLast()

O ES2023 adicionou o método findLast() que começa do final do array e retorna o valor do primeiro elemento que satisfaz a condição:

const temperaturas = [27, 28, 30, 40, 42, 35, 30];

// Encontra a última temperatura maior que 40
let alta = temperaturas.findLast(x => x > 40);
// Retorna 42

// Encontra a última temperatura maior que 30
let maior30 = temperaturas.findLast(x => x > 30);
// Retorna 35 (busca do final para o início)

Recurso ES2023: O JavaScript 2023 é suportado em todos os navegadores modernos desde Julho de 2023.

JavaScript Array findLastIndex()

O método findLastIndex() encontra o índice do último elemento que satisfaz a condição:

const temperaturas = [27, 28, 30, 40, 42, 35, 30];

// Encontra o índice da última temperatura maior que 40
let posicao = temperaturas.findLastIndex(x => x > 40);
// Retorna 4 (42 está no índice 4)

// Encontra o índice da última temperatura maior que 35
let pos2 = temperaturas.findLastIndex(x => x > 35);
// Retorna 4 (42 no índice 4 é maior que 35)

Comparação dos Métodos

Método Retorna Direção Usa Callback
indexOf() Índice Início → Fim Não
lastIndexOf() Índice Fim → Início Não
includes() Boolean Início → Fim Não
find() Valor Início → Fim Sim
findIndex() Índice Início → Fim Sim
findLast() Valor Fim → Início Sim
findLastIndex() Índice Fim → Início Sim
