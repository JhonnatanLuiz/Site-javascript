JavaScript Array Iteration

Métodos de Iteração de Array

Os métodos de iteração operam em cada item do array:

  • forEach() - Executa função para cada elemento
  • map() - Cria novo array transformado
  • flatMap() - Mapeia e achata
  • filter() - Filtra elementos
  • reduce() - Reduz a um único valor
  • reduceRight() - Reduz da direita
  • every() - Todos passam no teste?
  • some() - Algum passa no teste?
  • Array.from() - Cria array de iterável
  • keys() - Retorna chaves
  • entries() - Retorna pares
  • with() - Atualiza elemento (ES2023)

JavaScript Array forEach()

O método forEach() chama uma função (callback) uma vez para cada elemento do array:

const numeros = [45, 4, 9, 16, 25];
let texto = "";

numeros.forEach(function(valor, indice, array) {
    texto += valor + "<br>";
});

// Versão simplificada (usando apenas valor)
numeros.forEach(function(valor) {
    texto += valor + "<br>";
});

// Com arrow function
numeros.forEach(valor => texto += valor + "<br>");

A função callback recebe 3 argumentos:

  • O valor do item
  • O índice do item
  • O próprio array

JavaScript Array map()

O método map() cria um novo array executando uma função em cada elemento. Ele não modifica o array original:

const numeros1 = [45, 4, 9, 16, 25];

// Multiplicar cada valor por 2
const numeros2 = numeros1.map(function(valor) {
    return valor * 2;
});
// [90, 8, 18, 32, 50]

// Com arrow function
const numeros3 = numeros1.map(valor => valor * 2);

Nota: O método map() não executa a função para elementos vazios e não altera o array original.

JavaScript Array flatMap()

O ES2019 adicionou o método flatMap() que primeiro mapeia todos os elementos e depois achata o array resultante:

const meuArr = [1, 2, 3, 4, 5, 6];

// Cria pares [número, número * 10] e achata
const novoArr = meuArr.flatMap(x => [x, x * 10]);
// [1, 10, 2, 20, 3, 30, 4, 40, 5, 50, 6, 60]

JavaScript Array filter()

O método filter() cria um novo array com os elementos que passam em um teste:

const numeros = [45, 4, 9, 16, 25];

// Filtrar números maiores que 18
const maiores18 = numeros.filter(function(valor) {
    return valor > 18;
});
// [45, 25]

// Com arrow function
const maiores18b = numeros.filter(valor => valor > 18);

JavaScript Array reduce()

O método reduce() executa uma função em cada elemento para produzir um único valor. Ele trabalha da esquerda para a direita:

const numeros = [45, 4, 9, 16, 25];

// Somar todos os números
let soma = numeros.reduce(function(total, valor) {
    return total + valor;
});
// 99

// Com valor inicial
let soma2 = numeros.reduce(function(total, valor) {
    return total + valor;
}, 100);  // Começa com 100
// 199

// Com arrow function
let soma3 = numeros.reduce((total, valor) => total + valor, 0);

A função callback recebe 4 argumentos:

  • O total (valor inicial / valor retornado anteriormente)
  • O valor do item
  • O índice do item
  • O próprio array

Nota: O método reduce() não modifica o array original.

JavaScript Array reduceRight()

O método reduceRight() funciona igual ao reduce(), mas trabalha da direita para a esquerda:

const numeros = [45, 4, 9, 16, 25];

let soma = numeros.reduceRight(function(total, valor) {
    return total + valor;
});
// 99 (mesmo resultado, mas processando do final para o início)

JavaScript Array every()

O método every() verifica se todos os valores do array passam em um teste:

const numeros = [45, 4, 9, 16, 25];

// Todos são maiores que 18?
let todosMaiores18 = numeros.every(function(valor) {
    return valor > 18;
});
// false (4, 9, 16 não são maiores que 18)

// Todos são maiores que 0?
let todosMaioresZero = numeros.every(valor => valor > 0);
// true

JavaScript Array some()

O método some() verifica se algum valor do array passa em um teste:

const numeros = [45, 4, 9, 16, 25];

// Algum é maior que 18?
let algumMaior18 = numeros.some(function(valor) {
    return valor > 18;
});
// true (45 e 25 são maiores que 18)

// Algum é maior que 100?
let algumMaior100 = numeros.some(valor => valor > 100);
// false

JavaScript Array.from()

O método Array.from() cria um Array a partir de qualquer objeto iterável ou que tenha propriedade length:

// Criar array a partir de string
let texto = "ABCDEFG";
let arr = Array.from(texto);
// ["A", "B", "C", "D", "E", "F", "G"]

// Com função de transformação
const numeros = [1, 2, 3, 4];
const dobrados = Array.from(numeros, x => x * 2);
// [2, 4, 6, 8]

keys() e entries()

Array keys()

O método keys() retorna um Iterator com as chaves (índices) do array:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
const chaves = frutas.keys();

let texto = "";
for (let x of chaves) {
    texto += x + " ";
}
// "0 1 2 3"

Array entries()

O método entries() retorna um Iterator com pares [índice, valor]:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
const entradas = frutas.entries();

for (let x of entradas) {
    console.log(x);
}
// [0, "Banana"]
// [1, "Laranja"]
// [2, "Maçã"]
// [3, "Manga"]

JavaScript Array with() (ES2023)

O ES2023 adicionou o método with() como uma forma segura de atualizar um elemento sem alterar o array original:

const meses = ["Janeiro", "Fevereiro", "Mar", "Abril"];

// Corrigir "Mar" para "Março" no índice 2
const mesesCorrigidos = meses.with(2, "Março");

// meses: ["Janeiro", "Fevereiro", "Mar", "Abril"] (inalterado)
// mesesCorrigidos: ["Janeiro", "Fevereiro", "Março", "Abril"]

Spread (...) e Rest (...)

Operador Spread (...)

O operador spread expande um array em elementos individuais:

// Juntar arrays
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];
const arr3 = [...arr1, ...arr2];
// [1, 2, 3, 4, 5, 6]

// Copiar array
const copia = [...arr1];

// Passar como argumentos
const numeros = [23, 55, 21, 87, 56];
let minimo = Math.min(...numeros);  // 21
let maximo = Math.max(...numeros);  // 87

Operador Rest (...)

O operador rest coleta os elementos restantes em um array:

const arr = [1, 2, 3, 4, 5, 6, 7, 8];

// Desestruturar e coletar o resto
let [a, b, ...resto] = arr;
// a = 1
// b = 2
// resto = [3, 4, 5, 6, 7, 8]
