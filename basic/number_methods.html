JavaScript Number Methods

O método toString()

O método toString() retorna um número como uma string.

Exemplo

let x = 123;
x.toString();
(123).toString();
(100 + 23).toString();

Resultado:

O método toFixed()

toFixed() retorna uma string, com o número escrito com um número especificado de decimais:

Exemplo

let x = 9.656;
x.toFixed(0);
x.toFixed(2);
x.toFixed(4);
x.toFixed(6);

Resultado:

O método toPrecision()

toPrecision() retorna uma string, com um número escrito com um comprimento especificado:

Exemplo

let x = 9.656;
x.toPrecision();
x.toPrecision(2);
x.toPrecision(4);
x.toPrecision(6);

Resultado:

Convertendo Variáveis para Números

Existem 3 métodos JavaScript que podem ser usados para converter uma variável em um número:

  • Number()
  • parseInt()
  • parseFloat()

O método Number()

Number(true);
Number(false);
Number("10");
Number("  10");
Number("10  ");
Number(" 10  ");
Number("10.33");
Number("10,33");
Number("10 33");
Number("John");

Resultado:

O método parseInt()

parseInt("-10");
parseInt("-10.33");
parseInt("10");
parseInt("10.33");
parseInt("10 20 30");
parseInt("10 years");
parseInt("years 10");

Resultado:

Métodos de Objeto Number

Estes métodos pertencem ao objeto Number do JavaScript:

Number.isInteger()

Number.isInteger(10);
Number.isInteger(10.5);
