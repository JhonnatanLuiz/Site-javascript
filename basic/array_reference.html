JavaScript Array Reference

Esta página contém uma referência completa de todas as propriedades e métodos de Array em JavaScript.

Propriedades de Array

Propriedade Descrição
constructor Retorna a função que criou o protótipo do objeto Array
length Define ou retorna o número de elementos de um array
prototype Permite adicionar propriedades e métodos a um objeto Array

Métodos Básicos de Array

Método Descrição
at() Retorna um elemento em um índice especificado (permite índices negativos)
concat() Junta dois ou mais arrays e retorna um novo array
copyWithin() Copia elementos do array para outra posição no mesmo array
entries() Retorna um objeto Array Iterator com pares chave/valor
fill() Preenche os elementos de um array com um valor estático
flat() Cria um novo array com sub-arrays concatenados recursivamente
flatMap() Mapeia cada elemento e achata o resultado em um novo array
join() Junta todos os elementos em uma string
keys() Retorna um Array Iterator com as chaves (índices)
pop() Remove o último elemento e o retorna
push() Adiciona elementos ao final e retorna o novo tamanho
reverse() Inverte a ordem dos elementos no array
shift() Remove o primeiro elemento e o retorna
slice() Extrai uma porção do array e retorna um novo array
sort() Ordena os elementos do array
splice() Adiciona/remove elementos do array
toString() Converte o array em uma string e a retorna
unshift() Adiciona elementos no início e retorna o novo tamanho
values() Retorna um Array Iterator com os valores

Métodos de Busca

Método Descrição
find() Retorna o valor do primeiro elemento que passa no teste
findIndex() Retorna o índice do primeiro elemento que passa no teste
findLast() Retorna o valor do último elemento que passa no teste (ES2023)
findLastIndex() Retorna o índice do último elemento que passa no teste (ES2023)
includes() Verifica se o array contém um elemento especificado
indexOf() Retorna o primeiro índice de um elemento no array
lastIndexOf() Retorna o último índice de um elemento no array

Métodos de Iteração

Método Descrição
every() Verifica se todos os elementos passam no teste
filter() Cria um novo array com elementos que passam no teste
forEach() Executa uma função para cada elemento
map() Cria um novo array com resultados da função aplicada
reduce() Reduz o array a um único valor (da esquerda para direita)
reduceRight() Reduz o array a um único valor (da direita para esquerda)
some() Verifica se algum elemento passa no teste

Métodos Imutáveis (ES2023)

Estes métodos criam novos arrays sem modificar o original:

Método Descrição Equivalente Mutável
toReversed() Retorna novo array com ordem invertida reverse()
toSorted() Retorna novo array ordenado sort()
toSpliced() Retorna novo array com elementos alterados splice()
with() Retorna novo array com elemento alterado array[index] = value

Métodos Estáticos do Array

Método Descrição
Array.from() Cria um array a partir de um objeto iterável ou array-like
Array.isArray() Verifica se um valor é um array
Array.of() Cria um novo array a partir dos argumentos

Suporte por Versão ECMAScript

ES5 (2009)

  • • indexOf(), lastIndexOf()
  • • forEach(), map(), filter()
  • • reduce(), reduceRight()
  • • every(), some()
  • • Array.isArray()

ES6 (2015)

  • • find(), findIndex()
  • • Array.from(), Array.of()
  • • fill(), copyWithin()
  • • keys(), values(), entries()
  • • Spread operator (...)

ES2016+

  • • includes() (ES2016)
  • • flat(), flatMap() (ES2019)
  • • at() (ES2022)
  • • toSorted(), toReversed() (ES2023)
  • • toSpliced(), with() (ES2023)
