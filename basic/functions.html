JavaScript Functions

Uma função JavaScript é um bloco de código projetado para executar uma tarefa específica.

Uma função JavaScript é executada quando "algo" a invoca (chama).

Exemplo

// Função para calcular o produto de dois números
function myFunction(p1, p2) {
  return p1 * p2;
}

Sintaxe de Função JavaScript

Uma função JavaScript é definida com a palavra-chave function, seguida por um nome, seguido por parênteses ().

Os nomes de função podem conter letras, dígitos, sublinhados e cifrões (mesmas regras que variáveis).

Os parênteses podem incluir nomes de parâmetros separados por vírgulas:
(parametro1, parametro2, ...)

O código a ser executado pela função é colocado entre chaves: {}

Sintaxe

function nome(parametro1, parametro2, parametro3) {
  // código a ser executado
}

💡 Os parâmetros de função são listados entre parênteses () na definição da função.

💡 Os argumentos de função são os valores recebidos pela função quando ela é invocada.

💡 Dentro da função, os argumentos (os parâmetros) se comportam como variáveis locais.

Invocação de Função

O código dentro da função será executado quando "algo" invocar (chamar) a função:

  • Quando um evento ocorre (quando um usuário clica em um botão)
  • Quando é invocada (chamada) a partir do código JavaScript
  • Automaticamente (auto-invocada)

Exemplo

function toCelsius(fahrenheit) {
  return (5/9) * (fahrenheit-32);
}

let value = toCelsius(77);

Retorno de Função

Quando o JavaScript atinge uma declaração return, a função para de executar.

Se a função foi invocada a partir de uma declaração, o JavaScript "retornará" para executar o código após a declaração de invocação.

As funções geralmente calculam um valor de retorno. O valor de retorno é "retornado" de volta para o "chamador":

Exemplo

// Calcular o produto de dois números e retornar o resultado
function myFunction(a, b) {
  return a * b;
}

let x = myFunction(4, 3);

Por Que Usar Funções?

Com funções, você pode reutilizar código:

Defina o código uma vez e use-o muitas vezes.

Você pode usar o mesmo código com diferentes argumentos para produzir diferentes resultados.

Exemplo - Converter Fahrenheit para Celsius

function toCelsius(fahrenheit) {
  return (5/9) * (fahrenheit-32);
}

let text = "A temperatura é " + toCelsius(77) + " Celsius";

O Operador () Invoca a Função

Usando o exemplo acima, toCelsius refere-se ao objeto função, e toCelsius() refere-se ao resultado da função.

Acessar uma função sem () retorna a função e não o resultado da função:

Exemplo

function toCelsius(fahrenheit) {
  return (5/9) * (fahrenheit-32);
}

let text = toCelsius;

Funções Usadas como Valores de Variáveis

Funções podem ser usadas da mesma maneira que você usa variáveis, em todos os tipos de fórmulas, atribuições e cálculos.

Exemplo - Usando variável

let x = toCelsius(77);
let text = "A temperatura é " + x + " Celsius";

Você pode usar a função diretamente, como um valor de variável:

Exemplo - Usando função diretamente

let text = "A temperatura é " + toCelsius(77) + " Celsius";

Variáveis Locais

Variáveis declaradas dentro de uma função JavaScript se tornam LOCAIS para a função.

Variáveis locais só podem ser acessadas de dentro da função.

Exemplo

// código aqui NÃO PODE usar carName

function myFunction() {
  let carName = "Volvo";
  // código aqui PODE usar carName
}

// código aqui NÃO PODE usar carName

💡 Dica: Como as variáveis locais são reconhecidas apenas dentro de suas funções, variáveis com o mesmo nome podem ser usadas em diferentes funções.

💡 Dica: Variáveis locais são criadas quando uma função inicia e deletadas quando a função é concluída.
