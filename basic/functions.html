JavaScript Functions

Uma função JavaScript é um bloco de código projetado para executar uma tarefa específica.

Uma função JavaScript é executada quando "algo" a invoca (chama-a).

Exemplo

function myFunction(p1, p2) {
  return p1 * p2;   // A função retorna o produto de p1 e p2
}

Clique no botão para ver o resultado.

Sintaxe da Função

Uma função JavaScript é definida com a palavra-chave function, seguida por um nome, seguido por parênteses ().

Os nomes das funções podem conter letras, dígitos, sublinhados e cifrões (mesmas regras das variáveis).

Os parênteses podem incluir nomes de parâmetros separados por vírgulas:
(parâmetro1, parâmetro2, ...)

O código a ser executado pela função é colocado dentro de chaves: {}

function name(parameter1, parameter2, parameter3) {
  // código a ser executado
}

Os parâmetros da função são listados dentro dos parênteses na definição da função.

Os argumentos da função são os valores recebidos pela função quando ela é invocada.

Invocação de Função

O código dentro da função será executado quando "algo" invocar (chamar) a função:

  • Quando um evento ocorre (quando um usuário clica em um botão)
  • Quando é invocado (chamado) a partir do código JavaScript
  • Automaticamente (auto-invocado)

Retorno da Função

Quando o JavaScript atinge uma declaração return, a função para de executar.

Se a função foi invocada a partir de uma declaração, o JavaScript "retornará" para executar o código após a declaração de invocação.

Exemplo: Converter Fahrenheit para Celsius

function toCelsius(fahrenheit) {
  return (5/9) * (fahrenheit-32);
}

let value = toCelsius(77);

O Operador ()

O operador () invoca a função.

Acessar uma função sem () retornará a definição da função em vez do resultado da função.

Variáveis Locais

Variáveis declaradas dentro de uma função JavaScript tornam-se LOCAIS para a função.

Variáveis locais só podem ser acessadas de dentro da função.

Exemplo: Escopo Local

// código aqui NÃO pode usar carName

function myFunction() {
  let carName = "Volvo";
  // código aqui PODE usar carName
}

// código aqui NÃO pode usar carName

Resumo

  • Funções são blocos de código reutilizáveis.
  • Elas são definidas com a palavra-chave function.
  • Valores passados para funções são chamados de argumentos.
  • Funções podem retornar valores usando return.
  • Variáveis dentro de funções são locais e não acessíveis fora delas.
