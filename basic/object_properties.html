JavaScript Object Properties

Propriedades são a parte mais importante de qualquer objeto JavaScript.

Acessando Propriedades JavaScript

A sintaxe para acessar a propriedade de um objeto é:

objectName.property

ou

objectName["property"]

ou

objectName[expression]

Exemplo 1

person.firstname + " is " + person.age + " years old.";

Exemplo 2

person["firstname"] + " is " + person["age"] + " years old.";

Adicionando Novas Propriedades

Você pode adicionar novas propriedades a um objeto existente simplesmente atribuindo um valor a ela.

Suponha que o objeto person já exista - você pode dar a ele novas propriedades:

Exemplo

person.nationality = "English";

Deletando Propriedades

A palavra-chave delete deleta uma propriedade de um objeto:

Exemplo

const person = {
  firstName: "John",
  lastName: "Doe",
  age: 50,
  eyeColor: "blue"
};

delete person.age;

⚠️ A palavra-chave delete deleta tanto o valor da propriedade quanto a própria propriedade.

Após a exclusão, a propriedade não pode ser usada antes de ser adicionada novamente.

Objetos Aninhados

Valores em um objeto podem ser outro objeto:

Exemplo

myObj = {
  name: "John",
  age: 30,
  cars: {
    car1: "Ford",
    car2: "BMW",
    car3: "Fiat"
  }
}

Você pode acessar objetos aninhados usando a notação de ponto ou a notação de colchetes:

myObj.cars.car2;
ou:
myObj.cars["car2"];
ou:
myObj["cars"]["car2"];
