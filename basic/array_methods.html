JavaScript Métodos de Array

Métodos Básicos de Array

  • length - Tamanho do array
  • toString() - Converte para string
  • at() - Acessa elemento pelo índice
  • join() - Junta elementos com separador
  • pop() - Remove último elemento
  • push() - Adiciona ao final
  • shift() - Remove primeiro elemento
  • unshift() - Adiciona no início
  • concat() - Concatena arrays
  • splice() - Adiciona/remove elementos
  • slice() - Extrai parte do array
  • flat() - Achata arrays aninhados

JavaScript Array length

A propriedade length retorna o tamanho (quantidade de elementos) de um array:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
let tamanho = frutas.length;  // Retorna 4

A propriedade length também pode ser usada para definir o tamanho do array:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
frutas.length = 2;  // Agora frutas contém apenas ["Banana", "Laranja"]

JavaScript Array toString()

O método toString() converte um array em uma string separada por vírgulas:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
let texto = frutas.toString();
// Resultado: "Banana,Laranja,Maçã,Manga"

Nota: Todo objeto JavaScript tem um método toString(). O JavaScript converte automaticamente um array para string quando é necessário exibir como texto.

JavaScript Array at()

O ES2022 introduziu o método at() que retorna um elemento do array pelo índice:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];

// Usando at()
let fruta = frutas.at(2);  // "Maçã"

// Usando [] - mesmo resultado
let fruta2 = frutas[2];    // "Maçã"

O método at() permite índices negativos para acessar elementos do final:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
let ultimo = frutas.at(-1);  // "Manga" (último elemento)
let penultimo = frutas.at(-2);  // "Maçã"

Nota: Muitas linguagens permitem índices negativos como [-1] para acessar elementos do final. Isso não era possível em JavaScript com [], por isso o método at() foi introduzido no ES2022.

JavaScript Array join()

O método join() junta todos os elementos do array em uma string, permitindo especificar o separador:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];

let texto1 = frutas.join(" * ");
// Resultado: "Banana * Laranja * Maçã * Manga"

let texto2 = frutas.join(" - ");
// Resultado: "Banana - Laranja - Maçã - Manga"

Popping e Pushing

Quando trabalhamos com arrays, é fácil remover e adicionar elementos. Pop remove elementos do final, e Push adiciona ao final.

JavaScript Array pop()

O método pop() remove o último elemento de um array e retorna o valor removido:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
let removido = frutas.pop();  // Remove "Manga"
// frutas agora é ["Banana", "Laranja", "Maçã"]
// removido contém "Manga"

JavaScript Array push()

O método push() adiciona um novo elemento ao final do array e retorna o novo tamanho:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
let novoTamanho = frutas.push("Kiwi");  // Adiciona "Kiwi"
// frutas agora é ["Banana", "Laranja", "Maçã", "Manga", "Kiwi"]
// novoTamanho é 5

Shifting Elements

Shifting é equivalente ao popping, mas trabalha no primeiro elemento em vez do último.

JavaScript Array shift()

O método shift() remove o primeiro elemento do array e "desloca" todos os outros para um índice menor:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
let removido = frutas.shift();  // Remove "Banana"
// frutas agora é ["Laranja", "Maçã", "Manga"]
// removido contém "Banana"

JavaScript Array unshift()

O método unshift() adiciona um novo elemento no início do array e retorna o novo tamanho:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
let novoTamanho = frutas.unshift("Limão");  // Adiciona "Limão" no início
// frutas agora é ["Limão", "Banana", "Laranja", "Maçã", "Manga"]
// novoTamanho é 5

Array.isArray() e delete

Array.isArray()

O ECMAScript 5 adicionou o método Array.isArray() para verificar se um objeto é um array:

const frutas = ["Banana", "Laranja", "Maçã"];
Array.isArray(frutas);  // true

const texto = "Olá";
Array.isArray(texto);  // false

JavaScript Array delete()

Aviso! Usar delete deixa buracos undefined no array. Use pop() ou shift() em vez disso.

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
delete frutas[0];
// frutas agora é [undefined, "Laranja", "Maçã", "Manga"]
// O primeiro elemento virou undefined

JavaScript Array concat()

O método concat() cria um novo array mesclando (concatenando) arrays existentes:

// Mesclando dois arrays
const meninas = ["Cecília", "Joana"];
const meninos = ["Emílio", "Tomás", "Lucas"];
const criancas = meninas.concat(meninos);
// ["Cecília", "Joana", "Emílio", "Tomás", "Lucas"]

// Mesclando três arrays
const arr1 = ["Cecília", "Joana"];
const arr2 = ["Emílio", "Tomás"];
const arr3 = ["Rafael", "Marina"];
const todos = arr1.concat(arr2, arr3);

// Mesclando array com valores
const arr = ["Emílio", "Tomás", "Lucas"];
const novo = arr.concat("Pedro");
// ["Emílio", "Tomás", "Lucas", "Pedro"]

Nota: O método concat() não altera os arrays existentes. Ele sempre retorna um novo array.

Array copyWithin()

O método copyWithin() copia elementos do array para outra posição no mesmo array:

// Copiar para índice 2, todos os elementos do índice 0
const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
frutas.copyWithin(2, 0);
// ["Banana", "Laranja", "Banana", "Laranja"]

// Copiar para índice 2, elementos do índice 0 a 2
const frutas2 = ["Banana", "Laranja", "Maçã", "Manga", "Kiwi"];
frutas2.copyWithin(2, 0, 2);
// ["Banana", "Laranja", "Banana", "Laranja", "Kiwi"]

Nota: O método copyWithin() sobrescreve valores existentes, não adiciona itens e não altera o tamanho do array.

Achatando Arrays

"Achatar" um array é o processo de reduzir a dimensionalidade. É útil quando você quer converter um array multidimensional em um array unidimensional.

JavaScript Array flat()

O ES2019 introduziu o método flat() que cria um novo array com os sub-arrays concatenados até uma profundidade especificada:

const meuArr = [[1, 2], [3, 4], [5, 6]];
const novoArr = meuArr.flat();
// [1, 2, 3, 4, 5, 6]

// Com profundidade maior
const profundo = [1, [2, [3, [4]]]];
profundo.flat(1);  // [1, 2, [3, [4]]]
profundo.flat(2);  // [1, 2, 3, [4]]
profundo.flat(Infinity);  // [1, 2, 3, 4]

JavaScript Array flatMap()

O método flatMap() primeiro mapeia todos os elementos e depois cria um novo array achatando o resultado:

const meuArr = [1, 2, 3, 4, 5, 6];
const novoArr = meuArr.flatMap(x => [x, x * 10]);
// [1, 10, 2, 20, 3, 30, 4, 40, 5, 50, 6, 60]

JavaScript Array splice()

O método splice() pode ser usado para adicionar ou remover elementos em qualquer posição:

// Adicionando elementos
const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
frutas.splice(2, 0, "Limão", "Kiwi");
// ["Banana", "Laranja", "Limão", "Kiwi", "Maçã", "Manga"]

// Parâmetros:
// 1º (2): posição onde adicionar
// 2º (0): quantos elementos remover
// 3º...: novos elementos a adicionar

O método splice() retorna um array com os itens deletados:

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
let removidos = frutas.splice(2, 2, "Limão", "Kiwi");
// frutas: ["Banana", "Laranja", "Limão", "Kiwi"]
// removidos: ["Maçã", "Manga"]

Usando splice() para Remover

const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
frutas.splice(0, 1);  // Remove o primeiro elemento
// ["Laranja", "Maçã", "Manga"]

JavaScript Array toSpliced()

O ES2023 adicionou o método toSpliced() como uma forma segura de fazer splice sem alterar o array original:

const meses = ["Jan", "Fev", "Mar", "Abr"];
const alterado = meses.toSpliced(0, 1);
// meses: ["Jan", "Fev", "Mar", "Abr"] (inalterado)
// alterado: ["Fev", "Mar", "Abr"]

Diferença: O novo toSpliced() cria um novo array mantendo o original inalterado, enquanto splice() modifica o array original.

JavaScript Array slice()

O método slice() extrai uma parte do array e retorna um novo array, sem modificar o original:

const frutas = ["Banana", "Laranja", "Limão", "Maçã", "Manga"];

// Um argumento: do índice até o final
const citricos1 = frutas.slice(1);
// ["Laranja", "Limão", "Maçã", "Manga"]

// Dois argumentos: do início até o fim (não incluído)
const citricos2 = frutas.slice(1, 3);
// ["Laranja", "Limão"]

// O array original permanece inalterado
// frutas ainda é ["Banana", "Laranja", "Limão", "Maçã", "Manga"]

Nota: O método slice() cria um novo array e não remove elementos do array original. Se o argumento final for omitido, extrai até o final do array.
