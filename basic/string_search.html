JavaScript String Search

JavaScript oferece vários métodos para pesquisar strings:

  • String.indexOf()
  • String.lastIndexOf()
  • String.search()
  • String.match()
  • String.matchAll()
  • String.includes()
  • String.startsWith()
  • String.endsWith()

JavaScript String indexOf()

O método indexOf() retorna o índice (posição) da primeira ocorrência de uma string em uma string:

let texto = "Please locate where 'locate' occurs!";
let index = texto.indexOf("locate");

JavaScript String search()

O método search() pesquisa uma string por um valor especificado e retorna a posição da correspondência:

let texto = "Please locate where 'locate' occurs!";
let index = texto.search("locate");

JavaScript String includes()

O método includes() retorna true se uma string contiver um valor especificado.

let texto = "Hello world, welcome to the universe.";
let resultado = texto.includes("world");
